    An analytic solution of the volume rendering equation

    An analytic solution of the volume rendering equation

    November 7, 2020

    Solution of the Volume Rendering Equation #

    散乱媒質に対して, 輝度 \(L(\bm{x},\bm{\omega}_o,\nu)\) の光が方向 \(\bm{\omega}_i\) から入射し, 方向 \(\bm{\omega}_o\) へ散乱することを考える. この媒質の散乱係数を \(\mu_s\), 吸収係数を \(\mu_a\), 消散係数を \(\mu_t = \mu_a + \mu_s\) と書く. また散乱の位相関数を \(p(\bm{\omega}_i, \bm{\omega}_o)\), 自己発光を \(S(\bm{x},\bm{\omega}_o)\) とかく. このとき, Volume Rendering 方程式 は \[ \left(\bm{\omega}_o\cdot\nabla + \mu_t(\bm{x})\right)L(\bm{x},\bm{\omega}_o,\nu) = S(\bm{x},\bm{\omega}_o) + \mu_s(\bm{x})\int_{S^2}p(\bm{\omega}_i, \bm{\omega}_o)L(\bm{x},\bm{\omega}_i,\nu)d\bm{\omega}_i\] と与えられる.

    いま, パラメータ \(s\) を使って \(\bm{x}(s)=s\bm{\omega}\) と書けば, 右辺第1項は \[ \left(\bm{\omega}\cdot\nabla\right)L(\bm{x},\bm{\omega},\nu) = \left(\frac{d\bm{x}(s)}{ds}\cdot\nabla\right)L(\bm{x}(s),\bm{\omega},\nu) = \frac{dL(\bm{x}(s),\bm{\omega},\nu)}{ds}\] となる. これを Volume Rendering 方程式に適用すれば \[ \left(\frac{d}{ds} + \mu_t(s)\right)L(s,\bm{\omega}_o,\nu) = S(s,\bm{\omega}_o) + \mu_s(s)\int_{S^2}p(\bm{\omega}_i, \bm{\omega}_o)L(s,\bm{\omega}_i,\nu)d\bm{\omega}_i\] が得られる. ここで, \(L(s,\bm{\omega})=L(\bm{x}(s),\bm{\omega},\nu),\;,\mu(\bm{x}(s))=\mu(s),\; S(s,\bm{\omega})=S(\bm{x}(s),\bm{\omega})\) と略記した.

    上式を解けば, Volume Rendering 方程式の一般解 \[ L(s,\bm{\omega}_o) = T(0,s)L(0,\bm{\omega}_o) + \int^s_0 T(s',s)\left[S(s,\bm{\omega}_o) + \mu_s(s)L_s(s,\bm{\omega}_o)\right]ds',\] \[\begin{aligned} T(a,b) &= e^{-\int^b_a\mu_t(t)dt},\\ L_s(s,\bm{\omega}_o) &= \int_{S^2}p(\bm{\omega}_i, \bm{\omega}_o)L(s,\bm{\omega}_i,\nu)d\bm{\omega}_i, \end{aligned}\] が得られる.

    Uniform Media #

    媒質が一様であると仮定する. この場合, 散乱係数 \(\mu_s, \;\mu_a\) や自己発光 \(S\), 光の輝度 \(L(\bm{\omega})\) は位置に依存しない.

    このとき, Volume Rendering 方程式の解は \[ L(\bm{\omega}_o) = e^{-\mu_t s}L(0,\bm{\omega}_o) + \int^s_0 e^{-\mu_t(s-s')}\left[S(\bm{\omega}_o) + \mu_sL_s(\bm{\omega}_o)\right]ds',\] \[ L_s(\bm{\omega}_o) = \int_{S^2}p(\bm{\omega}_i, \bm{\omega}_o)L(\bm{\omega}_i)d\bm{\omega}_i\] となる. これを解けば, 一様媒質を通過した光線の輝度 \[\begin{aligned} L(\bm{\omega}_o) &= e^{-\mu_t s}L(0,\bm{\omega}_o) + \left[S(\bm{\omega}_o) + \mu_sL_s(\bm{\omega}_o)\right]e^{-\mu_ts} \int^s_0e^{\mu_ts'}ds'\\ &= L(0,\bm{\omega}_o)\alpha(s) + L_\text{media}(\bm{\omega}_o)\left(1-\alpha(s)\right) \end{aligned}\] が得られる. ここで \[\begin{aligned} \alpha(s) &= e^{-\mu_t s}\\ L_\text{media}(\bm{\omega}_o) &= \frac{S(\bm{\omega}_o) + \mu_sL_s(\bm{\omega}_o)}{\mu_t} \end{aligned}\] と置いた.

    Recurrence Relation #

    一様媒質に対する解は, 輝度 \(L(0,\bm{\omega}_o)\) の背景を, 透過率 \(\alpha\) と 輝度 \(L_\text{media}\) を持つ媒質を通して眺めることを表している.

    いま, この計算を繰り返し行うことで, 非一様な媒質を表現したい. \(n\)回目の計算距離を \(s^{(n)}\), 方向を \(\bm{\omega}^{(n)}\) として, \(\alpha^{(n)}:=\alpha(s^{(n)})\), \(L^{(n)}_\text{media}:=L_\text{media}(\bm{\omega}^{(n)}_o)\) と書く.

    このとき, 一様媒質を通過した光線の輝度は \[ L^{(n)}=L^{(n-1)}\alpha^{(n)} + L^{(n)}_\text{media}(1-\alpha^{(n)})\] という漸化式になる.

    計算を \(\bm{x}_1, \bm{x}_2, \dots, \bm{x}_N\) と \(N\) 回行うとき, この漸化式は \[\begin{aligned} L^{(1)} &= L^{(0)}\alpha^{(1)} + L^{(1)}_\text{media}(1-\alpha^{(1)})\\ L^{(2)} &= L^{(1)}\alpha^{(2)} + L^{(2)}_\text{media}(1-\alpha^{(2)})\\ &\vdots\\ L^{(N)} &= L^{(N-1)}\alpha^{(N)} + L^{(N)}_\text{media}(1-\alpha^{(N)}) \end{aligned}\] となる.

    ここで \[\begin{aligned} T^{(n)}&:=T^{(n-1)}\alpha^{(n)}, &T^{(0)}=1\\ I^{(n)}&:=I^{(n-1)}\alpha^{(n)} + L^{(n)}_\text{media}(1-\alpha^{(n)}),&I^{(0)}=0 \end{aligned}\] と置けば, 漸化式は \[\begin{aligned} L^{(1)} &= L^{(0)}\alpha^{(1)} + L^{(1)}_\text{media}(1-\alpha^{(1)})\\ &= L^{(0)}T^{(1)} + I^{(1)}\\ L^{(2)} &= L^{(1)}\alpha^{(2)} + L^{(2)}_\text{media}(1-\alpha^{(2)})\\ &= [L^{(0)}T^{(1)} + I^{(1)}]\alpha^{(2)} + L^{(2)}_\text{media}(1-\alpha^{(2)})\\ &= L^{(0)}\alpha^{(1)}\alpha^{(2)} + I^{(1)}\alpha^{(2)} + L^{(2)}_\text{media}(1-\alpha^{(2)})\\ &= L^{(0)}T^{(2)} + I^{(2)}\\ \vdots\\ L^{(N)} &= L^{(0)}T^{(N)} + I^{(N)} \end{aligned}\] と書き直せる.

    従って, レイマーチングの各ステップで \(T^{(n)},\;I^{(n)}\) を計算し, 上式によって計算結果を合成すれば, 非一様な媒質を通過する光を表現できる.
