    Geometrical Optics

    Geometrical Optics #

    Ray of Light #

    細く絞った光のビームを考え, その光が通った経路を光線と呼ぶ.

    ここで考えた光線は, のちに同じ波面を共有する光子の集団として正当化する(ToDo).

    Index of Refraction #

    光は, 真空中では速さ \(c\;[\text{m}/\text{s}]\) で空間を伝搬する. そして, 物質中を伝わる光の速さ \(v\;[\text{m}/\text{s}]\) は真空中よりも遅くなる. この速度の比 \[ n = \frac{c}{v}\quad\;[-]\] を屈折率と呼ぶ.

    物質中で光が遅くなる理由は, のちに誘電体の分極から説明する( Born and Wolf, 1999, Sec. 2.2)(ToDo).

    Optical Path Length #

    時間 \(t\;[\text{s}]\) のあいだに光が真空中を進む距離は \(ct\;[\text{m}]\), 屈折率 \(n\) の媒質中を進む距離は \(vt\;[\text{m}]\) となる. このとき, 媒質中を進む距離に屈折率を掛け, 真空中を進む距離に換算した量 \(nvt\;[\text{m}]\) を光路長と呼ぶ.

    光が進む媒質が不均質で, 場所によって屈折率 \(n(\bm{r})\)が異なるときは, 光路長は次のように書ける.
    光線の経路を3次元空間中の曲線として \(\bm{r}(t)=\begin{pmatrix}x(t)\\y(t)\\z(t)\end{pmatrix},\;(a\leq t\leq b)\) とし, その位置の屈折率を \(n(\bm{r})\) と書く. このとき, 光路長 \(S\) は \[ S = \int^b_a n(\bm{r})\left|\frac{d\bm{r}}{dt}\right|dt\] と計算できる.

    Fermat’s Principle #

    光線は, 光路長 \(S\) が停留値 \[ \delta S[\bm{r}(t)] = 0\] となる経路 \(\bm{r}(t)\)を進行する. これは, フェルマーの原理と呼ばれている ( Born and Wolf, 1999, 山本, 2014).

    Hamilton’s Principle #

    パラメータ \(t\) を \(z\)軸にとり, 光線の経路 \(\bm{r}\) を \[ \bm{r}(t)=\begin{pmatrix}\bm{q}(t) \\ t\end{pmatrix},\quad \bm{q}(t)=\begin{pmatrix}x(t) \\ y(t)\end{pmatrix}\] と表す. ここで, \(t\) による微分を \(\dot{\bm{q}} = \frac{d\bm{q}}{dt}\) と略記した.

    このとき, 光路長は \[ S[\bm{q},t] = \int^b_aL(\bm{q},\dot{\bm{q}},t)dt,\quad L(\bm{q},\dot{\bm{q}},t) = n(\bm{q},t)|\dot{\bm{r}}|\] となり, フェルマーの原理 は \[ \delta S[\bm{q},t] = 0\] と表せる. これは, 配位空間を2次元とする Hamiltonの原理に等しい.
